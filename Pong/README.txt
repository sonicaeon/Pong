/////////////////////////Installation/////////////////////////
Launch Pong.sln located in ../Pong/Pong.sln
Restore/download required NuGet packages (Bespoke.games.framework.1.4.0)

Build the Solution/Project, it should be able to build any
of the configuration modes (DEBUG/Release @ WIN32/x64).

//////////////////////Controls (Keyboard)//////////////////////

* Up/Down Arrow Keys - Move the paddle up and down respectively
* Left/Right Arrow Keys - Move the paddle up and down respectively
* ESC - Quit the game
